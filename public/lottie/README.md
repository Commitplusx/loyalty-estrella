# Lottie Animations — Estrella Delivery

Pega aquí los 3 archivos JSON de tus animaciones Lottie.

## Nombres exactos requeridos

| Archivo              | Paso  | Descripción                        |
|----------------------|-------|------------------------------------|
| `step-pedir.json`    | Paso 1 | Animación de pedir / teléfono     |
| `step-qr.json`       | Paso 2 | Animación del QR / escaneo        |
| `step-recibir.json`  | Paso 3 | Animación de recibir / regalo     |

## Cómo obtenerlos

1. Ve a https://lottiefiles.com
2. Busca la animación que quieras
3. Descárgala como **JSON (Lottie)**
4. Renómbrala con el nombre de la tabla de arriba
5. Pégala en esta carpeta

## Estructura final esperada

```
public/
  lottie/
    step-pedir.json      ✅
    step-qr.json         ✅
    step-recibir.json    ✅
    README.md
```

Una vez que los 3 archivos estén aquí, el stepper interactivo funcionará automáticamente.
